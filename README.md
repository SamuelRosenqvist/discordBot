# discordLoLBot

Micro project using discord Bot API.

## Available commands:
### `!aram <nbr-of-players> <rerolls-per-player/default: 2>` 
Randomizes picks for two teams, if uneven number of players assumes team 2 has the additional member.

e.g `!aram 6 2`
~~~~
Team1        Team2      
------------------------
Ivern        Zoe        
Gangplank    Seraphine  
Malzahar     Talon      
Zed          Mordekaiser
Poppy        Vi         
Renekton     Yone       
~~~~
